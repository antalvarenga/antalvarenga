# Contracts: Initial feature – Systems Dashboard Portfolio

This feature is a frontend-only single-page dashboard with no dedicated backend APIs introduced as part of the scope.

- No external HTTP APIs are required to render the dashboard.
- All data for panels (profile, skills, projects, capabilities, activity logs) is provided via static configuration in the frontend code.

If future features introduce backend services, their API contracts (e.g., OpenAPI or GraphQL schemas) should be added in this directory.
